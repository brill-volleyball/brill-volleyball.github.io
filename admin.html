Website Administration Guide

This page provides instructions on how to update your volleyball website with new tournaments, matches, photos, and videos. This guide is designed to be straightforward and requires no advanced technical skills.

1. Accessing Your Website Files

Your website consists of HTML files that you'll need to edit to add new content:

  • index.html - The home page
  • cva15.html - The 15U Black team page
  • cva13.html - The 13U Red team page
  • howardhigh.html - The Howard High School page
  • about.html - Your about page
  • styles.css - The styling file (you usually won't need to edit this)
  • script.js - The JavaScript file (you usually won't need to edit this)

You can edit these files using any text editor, such as:

  • Visual Studio Code (recommended, free download at code.visualstudio.com)
  • Notepad++ (Windows)
  • TextEdit (Mac)
  • Sublime Text

2. Adding a New Tournament to 15U Black Page

To add a new tournament to the 15U Black page:

  1. Open the cva15.html file in your text editor
  2. Locate the tournament table section (look for <table class="tournament-table">)
  3. Find the <tbody> tag within the table
  4. Add your new tournament by copying and pasting the template below as the first entry after the <tbody> tag:
<tr> <td>Tournament Name</td> <td>Month Day, Year</td> <td><a href="https://www.flickr.com/gp/197923315@N02/YOURLINK">Photos</a></td> <td><a href="https://www.youtube.com/playlist?list=YOURPLAYLISTID">Livestream Playlist</a></td> <td><a href="https://www.youtube.com/playlist?list=YOURPLAYLISTID">Balltime Playlist</a></td> </tr>

Replace:

  • Tournament Name with the actual tournament name
  • Month Day, Year with the actual date
  • YOURLINK with your Flickr album link ID
  • YOURPLAYLISTID with your YouTube playlist ID
Tip: If you don't have photos or videos for a particular category, replace the link with — like this: <td>—</td>

3. Adding a New Tournament to 13U Red Page

Follow the same process as for the 15U page, but edit the cva13.html file instead. The template for the 13U page is slightly different:

<tr> <td>Tournament Name</td> <td>Month Day, Year</td> <td><a href="https://www.flickr.com/gp/197923315@N02/YOURLINK">Photos</a></td> <td><a href="https://www.youtube.com/playlist?list=YOURPLAYLISTID">Video Playlist</a></td> </tr>
Tip: If you don't have photos for a tournament, you can use <td>N/A</td> as shown in some of your existing entries.

4. Adding a New Match to Howard High School Page

The process for adding a new match to the Howard High School page is similar:

  1. Open the howardhigh.html file in your text editor
  2. Locate the match table section
  3. Add your new match as the first entry in the table body:
<tr> <td>Opponent Name</td> <td>Month Day, Year</td> <td><a href="https://www.flickr.com/gp/197923315@N02/YOURLINK">Photos</a></td> </tr>

5. Updating the Home Page Featured Tournaments

To update the featured tournaments on the home page:

  1. Open the index.html file in your text editor
  2. Find the section with <section class="featured">
  3. Update the tournament information within this section
<div class="tournament"> <h3>Tournament Name</h3> <p>Month Day, Year</p> <div class="tournament-links"> <a href="https://www.flickr.com/gp/197923315@N02/LINKID" class="btn btn-small">15U Photos</a> <a href="https://www.youtube.com/playlist?list=PLAYLISTID" class="btn btn-small">15U Videos</a> <a href="https://www.flickr.com/gp/197923315@N02/LINKID" class="btn btn-small">13U Photos</a> <a href="https://www.youtube.com/playlist?list=PLAYLISTID" class="btn btn-small">13U Videos</a> </div> </div>

6. Updating Featured Video Playlists

To update the featured video playlists on any page:

  1. Find the iframe code in the respective HTML file
  2. Update the src attribute with your new YouTube playlist URL
<iframe width="560" height="315" src="https://www.youtube.com/embed/videoseries?list=YOURPLAYLISTID" title="YouTube video player" allow="accelerometer; autoplay; clipboard-write; encrypted-media; gyroscope; picture-in-picture; web-share" referrerpolicy="strict-origin-when-cross-origin" allowfullscreen></iframe>

Replace YOURPLAYLISTID with your actual YouTube playlist ID.

7. Getting Links from Flickr and YouTube

Here's how to get the correct links from your platforms:

Flickr Links:

  1. Upload your photos to Flickr and create an album
  2. Go to the album page
  3. Click the "Share" button
  4. Copy the link provided
  5. The format should be something like: https://www.flickr.com/gp/197923315@N02/UNIQUECODE

YouTube Playlist Links:

  1. Create and upload your videos to YouTube
  2. Create a playlist for the tournament
  3. Go to the playlist page
  4. Copy the URL from your browser's address bar
  5. Look for the "list=" part of the URL - that's your playlist ID
  6. For example, in https://www.youtube.com/playlist?list=PLnsC7rVeRHBsn84HyrJwZB5CIRbVWYHqn, the playlist ID is PLnsC7rVeRHBsn84HyrJwZB5CIRbVWYHqn

8. Testing Your Changes

Before uploading your changes to the live website:

  1. Save the HTML file you edited
  2. Open the file in your web browser (double-click the file or right-click and select "Open with" your browser)
  3. Check that your new entry appears correctly
  4. Test all the links to make sure they work
Important: Always make a backup copy of your HTML files before making changes. This way, if something goes wrong, you can restore the previous version.

9. Uploading Your Updated Files

After testing your changes, upload the updated HTML files to your web hosting service using the provided FTP credentials or website management tool.

If you're not familiar with this process, ask for assistance from whoever is hosting your website.

10. Template for New Season Setup

At the beginning of a new season, you'll want to create new pages or update the existing ones:

  1. Update the year in the page titles (e.g., "2025-2026 Season")
  2. Clear out the old tournament tables or create new pages
  3. Update any team information that has changed

It's recommended to keep the old season pages available by renaming them (e.g., cva15_2024-2025.html) and creating new pages for the current season.
